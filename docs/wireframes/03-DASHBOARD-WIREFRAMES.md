# 主仪表板界面线框图

## 📊 页面概览

仪表板是用户最常访问的页面，需要：
- **一目了然** - 关键信息立即可见
- **减少焦虑** - 积极展示，避免警报感
- **快速操作** - 常用功能一键触达
