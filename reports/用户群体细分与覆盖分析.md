## Savvly 当前目标用户群体

基于市场研究报告，Savvly 的核心定位是：

### 主要用户画像（Primary Persona）

**"自由职业的芙蕾雅"（Freelancer Freya）**

```
基本信息：
├─ 年龄：28 岁
├─ 职业：平面设计自由职业者
├─ 地点：温哥华
└─ 收入：$2,800-$4,500/月（可变）

财务状况：
├─ 收入模式：不规则（"盛宴或饥荒"）
├─ 税务：需自己预留 25-30%
├─ 应急基金：难以建立（目标 3-6 个月）
└─ 月度缓冲：$100-750（不稳定）

核心痛点：
├─ "某些月份赚很多但忘记预留税款"
├─ "账单月我恐慌"
├─ "我需要知道我实际有多少钱"
└─ 收入不可预测导致持续焦虑

心理需求：
├─ 掌控感（对财务的主动权）
├─ 希望（看到进展和未来）
└─ 焦虑缓解（平静而非警报）

使用习惯：
├─ 移动优先（手机为主）
├─ 每天检查（高频率）
└─ 需要快速洞察（无时间学习复杂系统）
```

**Savvly 为这类用户提供的核心价值**：
1. **收入平滑可视化**：将混乱的收入波动可视化为可理解的模式
2. **最低月预算**：基于最低收入月的安全预算
3. **自动税务预留**：每笔收入自动标记 25-30% 为税务
4. **应急基金优先**：突出显示 3-6 个月目标进展
5. **焦虑缓解设计**：温和色彩、积极框架、非评判语言

### Savvly 的核心市场定位

**差异化定位声明**：
> "Savvly 是为不可预测收入者设计的预算 App，提供焦虑缓解工具和希望驱动的可视化，在实现财务掌控的同时建立面向未来的信心。"

**三大支柱**：
1. **不规则收入专家**（vs 竞品的固定薪资假设）
2. **焦虑缓解设计**（vs 竞品的警报/惩罚式设计）
3. **希望可视化**（vs 竞品的过去支出聚焦）

**目标市场规模**：
- 44% 千禧一代从事副业/零工经济
- 68% 自由职业者报告收入不可预测困难
- 25-35 岁在高消费城市（温哥华、多伦多、旧金山等）
- 18-25 岁学生（辅助市场）

#### ✗ Savvly 不完全覆盖你的部分（30-40%）

**1. 储蓄消耗期专属功能（缺失）**

```
你的核心需求：
❌ 储蓄跑道计算器
   - "我的储蓄能撑多久？"
   - 当前 Savvly 设计聚焦"储蓄增长"而非"储蓄消耗"

❌ 负现金流可视化
   - "我的月度净消耗是否可控？"
   - Savvly 假设用户试图增加净资产，而非计划消耗

❌ 多阶段财务规划
   - 阶段 1：学生消耗期（当前）
   - 阶段 2：求职缓冲期（毕业后 1-3 月）
   - 阶段 3：收入积累期（找到工作后）
   - Savvly 未明确支持"生命周期阶段切换"

❌ 大额不规律支出优化
   - 学费（$5,000+ 每学期）
   - Savvly 的"真实开支"功能适用，但需调整为"从储蓄预留"而非"月度储蓄"
```

**市场空白**：
Savvly 当前定位在"不规则收入 + 试图积累储蓄"人群，但**你属于"不规则收入 + 计划性消耗储蓄"人群**。

这是一个**未被明确服务的子市场**：
- CS/MBA/医学等高投资学位的学生
- 职业转型期（辞职进修、创业准备期）
- Gap Year 旅行者（有储蓄支撑的短期消耗）
- 早退休人群（Fire 运动，但仍年轻）

## 全球预算 App 用户的完整细分

基于财务状况、收入模式、生命周期阶段，以下是完整的用户分类：

### 维度 1：按收入模式分类

#### 1. 固定收入者（Salaried）
```
特征：
├─ 每月固定薪水
├─ 可预测的现金流
└─ 雇主代扣税款和福利

子类型：
├─ 1A. 全职员工（高收入）：$60K-150K+/年
│   └─ 痛点：投资整合、税务优化、财富增长
├─ 1B. 全职员工（中等收入）：$40K-60K/年
│   └─ 痛点：储蓄困难、债务管理、应急基金
└─ 1C. 全职员工（低收入）：<$40K/年
    └─ 痛点：月光族、账单覆盖、食品不安全

代表性 App：
├─ Mint（已关闭，原服务此群体）
├─ Monarch Money（高收入 + 投资整合）
└─ PocketGuard（简单支出追踪）

市场份额：~50-60% 的预算 App 用户
```

---

#### 2. 不规则收入者（Irregular Income）
```
特征：
├─ 月度收入波动
├─ 需自己预留税款
└─ "盛宴或饥荒"周期

子类型：
├─ 2A. 自由职业者（稳定客户）
│   ├─ 收入：$3,000-6,000/月（可变但有底线）
│   └─ 痛点：税务预留、收入平滑、应急基金
│
├─ 2B. 零工经济（平台工作者）
│   ├─ 收入：$1,500-4,000/月（高度可变）
│   └─ 痛点：极度不可预测、无福利、多平台管理
│
├─ 2C. 佣金销售
│   ├─ 收入：基本工资 + 佣金
│   └─ 痛点：月度波动、年度规划困难
│
└─ 2D. 季节性工作
    ├─ 收入：夏季/假期高，其他时间低/零
    └─ 痛点：淡季生存、旺季储蓄纪律

代表性 App：
├─ Savvly（专为此群体设计）
├─ Emma（英国，预测承诺支出）
└─ YNAB（方法论适用但需手动调整）

市场份额：~25-30%（增长中，44% 千禧一代有副业）
```

---

#### 3. 混合收入者（Hybrid Income）
```
特征：
├─ 全职工资 + 副业/兼职
├─ 主要收入稳定，额外收入可变
└─ 通常试图增加总收入

子类型：
├─ 3A. 全职 + 副业（增收型）
│   ├─ 收入：$50K 工资 + $500-1,500/月副业
│   └─ 痛点：时间管理、副业税务、目标分配
│
└─ 3B. 全职 + 被动收入
    ├─ 收入：工资 + 租金/投资分红
    └─ 痛点：资产追踪、税务复杂性

代表性 App：
├─ Monarch Money（整合多收入来源）
├─ Quicken（复杂财务追踪）
└─ Savvly（如果副业收入不规则）

市场份额：~15-20%（增长中）
```

---

#### 4. 无收入/低收入者（No/Low Income）
```
特征：
├─ 失业、学生、退休
├─ 依赖储蓄、贷款、家人支持
└─ 负现金流或勉强平衡

子类型：
├─ 4A. 学生（消耗储蓄）
│   ├─ 收入：$0-1,500/月（兼职/助学金）
│   ├─ 支撑：个人储蓄/家人/贷款
│   └─ 痛点：储蓄跑道、学费规划、负现金流焦虑
│   ✨ 你在这里！
│
├─ 4B. 学生（完全依赖贷款）
│   ├─ 收入：$0（或最低兼职）
│   ├─ 支撑：学生贷款
│   └─ 痛点：债务累积、毕业后偿还恐惧
│
├─ 4C. 失业者（求职中）
│   ├─ 收入：失业金（如有）
│   ├─ 支撑：储蓄/遣散费
│   └─ 痛点：储蓄快速消耗、求职压力、焦虑
│
├─ 4D. 职业转型期
│   ├─ 收入：暂时放弃工资（进修/创业准备）
│   ├─ 支撑：过去储蓄
│   └─ 痛点：投资人力资本的不确定性
│
└─ 4E. 早退休（FIRE）
    ├─ 收入：投资收益（可能不稳定）
    ├─ 支撑：已积累资产
    └─ 痛点：资产提取策略、长期可持续性

代表性 App：
├─ YNAB（需调整为"消耗模式"）
├─ Goodbudget（信封法，手动为主）
└─ Savvly（部分适用，但缺少"消耗期"功能）

市场份额：~10-15%
市场服务：严重不足，现有 App 假设用户"试图积累"
```

### 维度 3：按财务健康状态分类

#### 类型 A：财务困境（Crisis）
```

├─ 入不敷出
├─ 高息债务（信用卡、发薪日贷款）
├─ 无应急基金
└─ 高财务焦虑

核心需求：
├─ 债务偿还计划（雪球/雪崩法）
├─ 削减支出指导
├─ 焦虑缓解
└─ 免费/低价工具

代表 App：
```

## 市场空白与机会

### 当前市场未被充分服务的群体

#### 1. 储蓄消耗期人群（High Priority Gap）

**包括**：
- ✨ **你这样的学生**（投资人力资本）
- 职业转型期（辞职进修、创业准备）
- 失业求职期（有储蓄缓冲）
- 新移民适应期（消耗原住国储蓄）
- Gap Year 旅行者（计划性消耗）
- 早退休试验期（测试 FIRE 可行性）

**市场规模**：
- 估计 5-10% 的预算 App 潜在用户
- 北美每年 ~200 万研究生/专业学位学生
- 职业转型者（难以统计，但增长趋势）

**核心需求**：
```
不是"如何存更多钱"，而是：
├─ "我的储蓄能撑多久？"（储蓄跑道）
├─ "我的消耗速度是否可控？"（净消耗追踪）
├─ "大额支出如何规划？"（学费、搬家等）
├─ "何时需要采取行动？"（触发规则）
└─ "这个投资何时回本？"（未来收入预测）
```

**为什么现有 App 不适合**：
- YNAB：假设用户试图"给每块钱分配工作"并积累
- Monarch/Empower：聚焦投资增长，不是消耗管理
- PocketGuard：聚焦日常支出，无长期跑道可视化
- Savvly：聚焦不规则收入积累，而非计划性消耗

**机会**：
开发"储蓄跑道模式"（Runway Mode）
- 一键切换："积累模式" ↔ "消耗模式"
- 消耗模式核心指标：
  - 储蓄跑道（月数）
  - 月度净消耗
  - 大额支出倒计时
  - 未来收入预测（如毕业后薪资）

---

#### 2. 极度不规则收入者（Savvly 部分覆盖，但可深化）

**包括**：
- 艺术家、音乐家（项目制收入）
- 初创公司创始人（早期无工资或极低）
- 房地产经纪人（佣金集中在几个大单）
- 季节性导游、教练

**核心需求**：
```
比典型自由职业者更极端：
├─ 可能连续 3-6 个月零收入
├─ 然后一次性大额收入
├─ 需要"储蓄池"管理而非"月度预算"
└─ 心理支持：长期零收入的焦虑缓解
```

**Savvly 可以深化的功能**：
- "丰收月"自动分配规则（如：>$5K 自动 50% 存应急）
- 零收入月的心理支持（"这是正常的，你的跑道还有 X 月"）
- 项目制收入追踪（按项目而非按月）

#### 3. 高收入但高支出者（"金手铐"群体）

**包括**：
- 医生、律师、科技高管（$150K-500K+/年）
- 但生活方式膨胀严重
- 感觉"收入高但存不下钱"

**核心需求**：
```
不是缺钱，而是缺控制：
├─ 可见性："钱都去哪了？"
├─ 对比：收入 vs 储蓄率（"我赚这么多为什么只存 5%？"）
├─ 生活方式审计：识别"隐形超支"
└─ 同侪对比：匿名对标（"你的储蓄率低于同收入群体"）
```

**为什么现有 App 不适合**：
- 大多数 App 聚焦"预算紧张"人群
- 高收入者觉得"预算"是给穷人用的（心理障碍）
- 需要"财富优化"而非"预算限制"框架

**机会**：
品牌为"财富优化工具"而非"预算 App"
- 强调效率而非节省
- 对标同收入群体（激发竞争心理）
- 整合税务优化、慈善捐赠追踪

**包括**：

**市场规模**：

**机会**：

- 北美缺少针对移民的本地化财务 App

**市场规模**：
- ADHD：美国 ~10% 成年人
- 严重服务不足，巨大机会
