## 🎨 设计原则回顾

### 核心价值观
- **赋权而非限制** - 积极语言，庆祝进展
- **希望而非恐惧** - 平静蓝绿色系，避免刺眼红色
- **平静而非警报** - 温和提醒，非焦虑触发
- **清晰而非复杂** - 一目了然，渐进披露
- **同理心而非评判** - "让我们调整一下"，非"你失败了"

### 色彩系统
```css
主色调：
- Primary Blue: #3B82F6 (信任、稳定)
- Secondary Green: #10B981 (成长、希望)
- Accent Orange: #F59E0B (乐观、温暖)
- Attention Coral: #FB923C (温和提醒，替代红色)
- Neutral Gray: #6B7280 (平静、专业)
```

### 字体系统
```
- 主字体: Inter (屏幕阅读优化)
- 关键数字: text-4xl (36px)
- 卡片标题: text-lg (18px)
- 正文: text-base (16px)
- 说明: text-sm (14px)
```

## 📐 响应式断点

### 设计适配策略
```
移动端 (< 640px):
- 单列布局
- 底部导航栏
- 全屏卡片
- 简化信息层级

平板 (640px - 1024px):
- 双列网格
- 侧边导航栏
- 部分卡片并排

桌面 (> 1024px):
- 三列网格
- 固定侧边栏
- 更多信息展示
- 悬浮交互效果
```

### Tailwind 断点
```css
/* 移动优先设计 */
默认: < 640px
sm: 640px   (平板竖屏)
md: 768px   (平板横屏)
lg: 1024px  (桌面)
xl: 1280px  (大屏桌面)
2xl: 1536px (超大屏)
```

## 📊 数据展示规范

### 货币格式
```javascript
// 正确格式
$1,800.00   // 大额显示小数
$45         // 小额可省略小数

// 语言友好
"本月可用收入: $1,800"  // 而非 "Available: $1800"
```

### 百分比格式
```javascript
72%    // 预算使用率
+15%   // 同比增长（绿色）
-8%    // 同比下降（橙色，非红色）
```

### 日期格式
```javascript
"10月27日"           // 短日期
"2025年10月27日"     // 完整日期
"3天后"              // 相对时间（账单到期）
"刚刚", "5分钟前"    // 交易时间
```

## 🎭 状态与交互

### 按钮状态
```
默认 → 悬停 → 点击 → 加载中 → 成功/错误
```

### 表单状态
```
空白 → 聚焦 → 输入中 → 验证 → 提交
  │                      │
  │                      ├─ ✓ 成功（绿色）
  │                      └─ ✗ 错误（温和橙色）
```

### 页面状态
```
加载中 → 数据加载 → 展示内容
   │                    │
   │                    ├─ 有数据：正常展示
   │                    └─ 无数据：友好空状态
   │
   └─ 错误：温和错误提示 + 重试按钮
```

## 🎨 微交互设计

### 动画时长
```css
快速反馈: 150ms  (按钮悬停、颜色变化)
标准过渡: 300ms  (卡片展开、模态框出现)
复杂动画: 500ms  (进度条填充、数字滚动)
庆祝动画: 1000ms (目标达成 confetti)
```

### 缓动函数
```css
ease-in-out   // 标准（最常用）
ease-out      // 入场（模态框）
ease-in       // 离场（关闭动画）
spring        // 弹性（庆祝效果）
```
